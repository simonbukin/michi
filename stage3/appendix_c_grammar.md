# Appendix C — Stage 3 Grammar Index

This appendix lists every grammar point introduced in Stage 3 (Chapters 1–27), organized by chapter. Each entry includes the formation rule, meaning, and one example sentence.

---

## How to Read This Index

**Columns:**

- **Grammar Point** — The pattern as it appears in instruction.
- **Formation** — How to construct the pattern from the base form.
- **Meaning** — English translation or functional description.
- **Example** — A model sentence demonstrating the pattern.
- **Ch.** — Chapter of introduction.

**Notes:** Some grammar points build on patterns introduced in Stages 1 and 2 (marked "review + expansion"). Cross-references are provided where patterns interact.
