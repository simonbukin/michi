# Appendix A — Stage 4 Vocabulary Index

This appendix consolidates vocabulary introduced across all Stage 4 chapters into a single alphabetical reference. Each entry includes the word, reading, pitch accent, part of speech, English meaning, and the chapter where it first appears. The list is organized by the Japanese syllabary rows (五十音順) for efficient lookup.

Pitch accent is marked with circled numbers: ⓪ = flat (heiban), ① = accent on mora 1, ② = accent on mora 2, and so on.

Abbreviations used in the 品詞 column: 名 = 名詞 (noun), 動五 = 五段動詞, 動一 = 一段動詞, ナ = な-adjective, イ = い-adjective, 副 = 副詞 (adverb), 接 = 接続詞 (conjunction), 擬 = 擬音語・擬態語 (onomatopoeia/mimetic word), 連 = 連語 (set phrase).

## Notes on Using This Index

1. **Chapter numbers** refer to Stage 4 chapters. Ch 1 = Stage 4, Chapter 1 (Classical Residues).
2. **Onomatopoeia** (擬) from Chapters 16--17 are included in the main alphabetical listing and are among the most distinctive vocabulary of Stage 4. Many can function as adverbs (with と or without), as する verbs, or as modifiers.
3. **Pitch accent** notation follows the standard NHK system. ⓪ indicates flat/heiban (no drop). ① through ⑨ indicate the mora after which pitch drops.
4. This index is a **representative selection**, not an exhaustive list of every word appearing in Stage 4. For complete chapter vocabulary, consult the vocabulary list at the end of each chapter.
