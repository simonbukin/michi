# Appendix A — Stage 5 Vocabulary Index

This appendix consolidates vocabulary introduced across all Stage 5 chapters into a single alphabetical reference. Each entry includes the word, reading, pitch accent, part of speech, English meaning, and the chapter where it first appears. The list is organized by the Japanese syllabary rows (五十音順) for efficient lookup.

Pitch accent is marked with circled numbers: ⓪ = flat (heiban), ① = accent on mora 1, ② = accent on mora 2, and so on.

Abbreviations used in the 品詞 column: 名 = 名詞 (noun), 動五 = 五段動詞, 動一 = 一段動詞, ナ = な-adjective, イ = い-adjective, 副 = 副詞 (adverb), 接 = 接続詞 (conjunction), 擬 = 擬音語・擬態語 (onomatopoeia/mimetic word), 連 = 連語 (set phrase), 感 = 感動詞 (interjection), 助 = 助詞 (particle), 古 = 古語 (classical term), 俗 = 俗語 (slang/colloquial).

## Notes on Using This Index

1. **Chapter numbers** refer to Stage 5 chapters. Ch 1 = Stage 5, Chapter 1 (Emphatic Assertion).
2. **Onomatopoeia** (擬) from Chapter 9 are included both in the main alphabetical listing and in a dedicated supplementary section for quick reference.
3. **Pitch accent** notation follows the standard NHK system. ⓪ indicates flat/heiban (no drop). ① through ⑤ indicate the mora after which pitch drops.
4. **Classical terms** (古) and **slang/colloquial terms** (俗) are marked in the part-of-speech column. Classical terms appear primarily in Chapters 10--11, while internet and fandom slang concentrates in Chapters 18--20.
5. **N1 grammar patterns** are listed in a dedicated supplementary section because they function as multi-word expressions rather than single vocabulary items.
6. This index is a **representative selection**, not an exhaustive list of every word appearing in Stage 5. For complete chapter vocabulary, consult the vocabulary list at the end of each chapter.
