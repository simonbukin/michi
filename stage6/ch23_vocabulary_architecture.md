# Chapter 23 — Vocabulary Architecture: Building from Kanji

At this level, the most powerful tool for vocabulary expansion is not memorization — it is structural analysis. Japanese vocabulary, particularly the Sino-Japanese (漢語) layer, is built from kanji in systematic, predictable ways. A single high-yield kanji can generate ten, twenty, or more compounds, and understanding the kanji's core meaning allows you to deduce the meaning of unfamiliar compounds on sight.

This chapter teaches you to think of kanji not as characters to memorize but as building blocks in an architecture. Once you understand how 制 contributes "control/system" to every compound it appears in, you can encounter 制裁, 制覇, 制約, or 制圧 for the first time and have a reasonable guess at the meaning before reaching for a dictionary.

The chapter also covers the exceptions: 熟字訓 (compounds with readings that bear no relation to the individual kanji readings), 当て字 (kanji used purely for their sound), and multi-reading kanji whose correct reading depends on context. These exceptions are finite and learnable, and knowing them removes the last major obstacles to fluent kanji reading.

## 23.1 漢字語構成 as Vocabulary Strategy

### How Sino-Japanese compounds work

Most two-kanji Sino-Japanese compounds (熟語) follow predictable structural patterns:

### Compound structures

| Structure | Pattern | Example | Analysis |
|---|---|---|---|
| Modifier + Modified | A describes B | 新聞 (new + hear = newspaper) | A modifies B |
| Verb + Object | A acts on B | 読書 (read + book = reading) | A is the action, B is the object |
| Subject + Predicate | A does B | 地震 (ground + shake = earthquake) | A is the subject, B is the action |
| Synonymous pair | A ≈ B | 道路 (road + road = road) | Both kanji mean roughly the same thing |
| Antonymous pair | A ↔ B | 売買 (sell + buy = trade) | Opposite meanings combine into a broader concept |
| Prefix + Base | A qualifies B | 不安 (not + peace = anxiety) | A is a prefix that modifies B |
| Base + Suffix | A + qualifier B | 科学的 (science + -like = scientific) | B is a suffix that categorizes A |

Understanding these patterns means that when you encounter an unknown compound, you can decompose it:

> 制御 = 制 (control) + 御 (manage/control) → "control" (synonymous pair: both kanji mean control)
> 制限 = 制 (control) + 限 (limit) → "restriction" (verb + object: to control by limiting)
> 制裁 = 制 (control/sanction) + 裁 (judge) → "sanctions" (verb + object: to judge and control)

### The core meaning principle

Each kanji carries a core meaning that appears in every compound it forms. This core meaning is sometimes concrete, sometimes abstract, but it is remarkably consistent:

- 制 → core meaning: "control, regulate, system"
- 構 → core meaning: "construct, build, structure"
- 適 → core meaning: "fit, suitable, match"
- 導 → core meaning: "lead, guide, conduct"

Once you identify the core meaning, you can navigate all compounds built from that kanji.

## 23.2 Worked Example: 制

### Core meaning: control, regulate, system

制 appears in compounds related to systems of control, regulation, restriction, and the establishment of rules. It is one of the highest-yield kanji in Japanese vocabulary.

| 強制 | きょうせい | ⓪ | compulsion, forcing | strong + control = forced compliance |
| 制作 | せいさく | ⓪ | production, creation | control + make = controlled making |
| 制定 | せいてい | ⓪ | enactment, establishment | control + fix = establishing by authority |
| 制覇 | せいは | ① | domination, conquest | control + supremacy = total control |
| 制約 | せいやく | ⓪ | constraint | control + bind = binding limitation |
| 制圧 | せいあつ | ⓪ | suppression (military) | control + pressure = forceful control |

| 統制 | とうせい | ⓪ | regulation, control | unify + control = coordinated control |
| 牽制 | けんせい | ⓪ | check, restraint | pull + control = holding in check |
| 先制 | せんせい | ⓪ | preemptive | first + control = striking first |

### Patterns within the network

Notice how the position of 制 in the compound shifts its role:

- **制 as first element (制 + X):** The control is active — something is being controlled, created, or established. 制限, 制御, 制作, 制裁, 制圧, 制定, 制覇, 制服, 制約.
- **制 as second element (X + 制):** The compound describes a type or system of control. 規制, 抑制, 強制, 体制, 自制, 統制, 牽制, 先制.

This positional pattern holds for many kanji: when a kanji appears first, it tends to be the active element; when it appears second, it tends to describe the nature or type of the concept.

### Example sentences

> 政府は新たな規制を導入した。
> "The government introduced new regulations."

> この制度には多くの制約がある。
> "This system has many constraints."

> 暴動は数時間で制圧された。
> "The riot was suppressed within a few hours."

> 彼は驚くべき自制心を持っている。
> "He has remarkable self-control."

## 23.4 Reading Strategy: Deducing Unknown Compounds

### The decomposition method

When you encounter an unknown compound in reading, follow these steps:

1. **Identify each kanji's core meaning.** If you know the kanji, recall its most common meaning. If you do not know it, look at the radical for a domain hint.

2. **Determine the compound structure.** Is it modifier + modified? Verb + object? Synonym pair? Antonym pair?

3. **Generate a hypothesis.** Combine the meanings according to the structure.

4. **Check against context.** Does your hypothesis make sense in the sentence?

### Worked examples

**Example 1:** You encounter 概算 for the first time.

- 概 = general, approximate (from 概念 = concept, 概要 = outline)
- 算 = calculate (from 計算 = calculation, 算数 = arithmetic)
- Structure: modifier + modified (approximate + calculation)
- Hypothesis: "rough calculation, estimate"
- Actual meaning: rough estimate, approximation. Correct.

**Example 2:** You encounter 遮光 for the first time.

- 遮 = block, obstruct (from 遮断 = cutoff)
- 光 = light (basic kanji)
- Structure: verb + object (block + light)
- Hypothesis: "blocking light, light-blocking"
- Actual meaning: light blocking, shading. Correct.

**Example 3:** You encounter 緻密 for the first time.

- 緻 = fine, detailed (from the radical 糸 = thread, suggesting fineness)
- 密 = dense, close (from 秘密 = secret, 密接 = close)
- Structure: synonymous pair (fine + dense)
- Hypothesis: "very detailed, precise"
- Actual meaning: minute, elaborate, meticulous. Correct.

**Example 4:** You encounter 遡及 for the first time.

- 遡 = trace back (辶 radical = movement; 朔 = beginning of a cycle)
- 及 = reach (from 及ぶ = to reach, 普及 = spread)
- Structure: verb + complement (trace back + reaching)
- Hypothesis: "reaching back, retroactive"
- Actual meaning: retroactive effect. Correct.

### When decomposition fails

Decomposition does not always work. Some compounds have meanings that have drifted from their components:

- 大丈夫 (だいじょうぶ) = "all right, fine" — not "big strong man" (though that is the etymological origin)
- 勉強 (べんきょう) = "study" — not "forced strength" (though the original meaning was closer to "making an effort")
- 結構 (けっこう) = "fine, quite" — the connection between "tie + build" and "that's fine" is historical, not transparent

These exceptions are in the minority. For most compounds — especially formal, academic, and technical vocabulary — decomposition is reliable.

## 23.5 熟字訓 — Irregular-Reading Compounds

### What are 熟字訓?

熟字訓 (じゅくじくん) are compounds where the reading is assigned to the whole word, not to the individual kanji. You cannot determine the reading by combining the on'yomi or kun'yomi of each kanji. The kanji represent the meaning; the reading is an existing Japanese word mapped onto those kanji.

For example, 大人 is read おとな, not だいじん or おおひと. The kanji 大 (big) and 人 (person) convey the meaning "adult," and the native Japanese word おとな is mapped onto those kanji as a unit.

### Why they exist

Japanese originally had no writing system. When kanji were imported from China, existing Japanese words were mapped onto kanji that conveyed the same meaning. Sometimes a two-kanji meaning unit corresponded to a single Japanese word, creating a mismatch between the kanji and the reading.

## 23.6 当て字 — Kanji Used Phonetically

### What are 当て字?

当て字 (あてじ) are kanji chosen for their sound rather than their meaning. The individual kanji have no semantic relationship to the word's meaning — they are purely phonetic. This is the opposite of 熟字訓, where the kanji carry the meaning and the reading is independent.

|--------|------|------|---------------------------|
| 珈琲 | コーヒー | coffee | jeweled hairpin + not |
| 倶楽部 | クラブ | club | together + pleasure + part |
| 浪漫 | ロマン | romance, romanticism | wave + aimless |
| 型録 | カタログ | catalog | type + record |
| 煙草 | たばこ | tobacco | smoke + grass |
| 硝子 | ガラス | glass | nitrate + child |
| 麦酒 | ビール | beer | wheat + alcohol |
| 葡萄 | ぶどう | grape | (both characters are phonetic) |
| 檸檬 | レモン | lemon | (both characters are phonetic) |
| 林檎 | りんご | apple | forest + crab apple |
| 合羽 | カッパ | raincoat (from Portuguese "capa") | join + feather |
| 天麩羅 | てんぷら | tempura | heaven + wheat + thin |

### Hybrid cases

Some words are partly semantic, partly phonetic:

- 煙草 (たばこ): 煙 (smoke) is semantic; 草 (grass) is semantic too — tobacco is a plant you smoke. But the reading たばこ is from Portuguese "tabaco," making this a foreign loanword written in semantically relevant kanji.
- 麦酒 (ビール): 麦 (wheat/barley) and 酒 (alcohol) are both semantically relevant — beer is a barley-based alcohol. But the reading ビール is from Dutch "bier."

### Modern 当て字

Contemporary Japanese occasionally creates playful 当て字:

- 本気 read as マジ in manga/light novels (meaning "for real")
- 友達 read as なかま or ダチ in fiction
- 運命 read as さだめ in song lyrics

These are creative readings (ルビ) that layer additional meaning: the kanji convey the written meaning while the furigana reading conveys the spoken/emotional meaning.

## 23.7 Multi-Reading Kanji Mastery

### The problem

Some kanji have many possible readings. The kanji 生, for instance, has over ten common readings. How do you know which one applies?

The answer is: context and compound. Each reading is associated with specific words, compounds, or grammatical positions. Once you learn which reading goes with which word, the "multiple readings" problem dissolves into a finite set of known items.

### 生 — The most versatile kanji

| 読み | Context | Examples |
|------|---------|---------|
| セイ | On'yomi — most Sino-Japanese compounds | 生活 (せいかつ, life), 学生 (がくせい, student), 生産 (せいさん, production), 先生 (せんせい, teacher), 誕生 (たんじょう, birth) |
| ショウ | On'yomi — birth/life compounds | 一生 (いっしょう, lifetime), 生涯 (しょうがい, lifelong) |
| いきる | Kun'yomi — "to live" | 生きる (いきる), 生き物 (いきもの, living thing) |
| いかす | Kun'yomi — "to make use of" | 生かす (いかす), 経験を生かす |
| いける | Kun'yomi — flower arrangement | 生ける (いける), 生け花 (いけばな) |
| うまれる | Kun'yomi — "to be born" | 生まれる (うまれる), 生まれ故郷 |
| うむ | Kun'yomi — "to give birth" | 生む (うむ), 子を生む |
| はえる | Kun'yomi — "to grow (plants/hair)" | 生える (はえる), 草が生える |
| はやす | Kun'yomi — "to grow (intentionally)" | 生やす (はやす), 髭を生やす |
| き | Kun'yomi — "raw, pure, unprocessed" | 生ビール (なまビール, draft beer) — wait, this is なま |
| なま | Kun'yomi — "raw, fresh, live" | 生卵 (なまたまご, raw egg), 生放送 (なまほうそう, live broadcast) |
| お（う） | Name reading | 生田 (いくた), 桐生 (きりゅう) — names have unpredictable readings |
| ふ | Name reading | 生田 (いくた) uses いく, but 芝生 uses しばふ |

**Strategy:** Do not try to memorize a table of readings. Instead, learn each word individually. You already know 学生, 生活, 生きる, 生まれる, 先生, 生ビール as separate vocabulary items. The "multiple readings" are already in your head as individual words — you just need to recognize that they all share one kanji.

### 下 — Position and direction

| 読み | Context | Examples |
|------|---------|---------|
| カ / ゲ | On'yomi | 地下 (ちか, underground), 下品 (げひん, vulgar), 下旬 (げじゅん, last third of month) |

### 日 — Time and light

| 読み | Context | Examples |
|------|---------|---------|
| ニチ / ジツ | On'yomi | 日曜日 (にちようび), 毎日 (まいにち), 本日 (ほんじつ), 祝日 (しゅくじつ) |
| ひ / び | Kun'yomi — "day, sun" | 今日 (きょう — 熟字訓), 日 (ひ, day), 誕生日 (たんじょうび) |
| か | Counter for days | 二日 (ふつか), 三日 (みっか), 四日 (よっか)... |

**Strategy for ニチ vs ジツ:** ニチ is the default. ジツ appears in formal/literary contexts: 本日 (ほんじつ), 休日 (きゅうじつ), 祝日 (しゅくじつ), 実日数 (じつにっすう). When in doubt, ニチ is usually correct.

### General strategy for multi-reading kanji

1. **Learn words, not readings.** You do not need to memorize that 生 has twelve readings. You need to know that 生活 is せいかつ and 生まれる is うまれる. The readings are a byproduct of knowing the words.

2. **On'yomi for compounds, kun'yomi for standalone use.** This rule covers about 80% of cases. When a kanji appears in a two-kanji compound, it usually takes its on'yomi. When it appears alone or with okurigana, it usually takes its kun'yomi.

3. **Common pairs are stable.** Certain reading patterns are stable across many kanji: 上がる/上げる, 下がる/下げる, 集まる/集める follow the intransitive/transitive pattern. Once you learn the pattern, new instances are predictable.

4. **Names are unpredictable.** Accept that personal and place names may use readings you have never seen. This is normal for native Japanese readers too.

## 23.9 Reading Passage: Vocabulary Architecture in Action

The following passage contains multiple compounds built from the kanji studied in this chapter. As you read, notice how compound decomposition allows you to process vocabulary efficiently.

> 日本の半導体産業は、かつて世界を制覇していた。しかし、1990年代以降、海外企業との競争に適応できず、主導権を失った。政府は規制緩和と新たな制度設計によって産業構造の転換を図ったが、構想は壮大だったものの、実践が追いつかなかった。制約の多い国内市場に縛られたまま、最適な戦略を構築できなかったのだ。
