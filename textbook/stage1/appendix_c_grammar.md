## Verb Forms

### Verb Classes (Ch 8)

Japanese has three verb classes. The class determines all conjugation patterns.

| Class | Name | Identification | Examples |
|---|---|---|---|
| 五段 | Consonant-stem | Dictionary form ends in う-row sound (not る preceded by い/え sound) | かく, のむ, はなす |
| 一段 | Vowel-stem | Dictionary form ends in る preceded by い or え sound | たべる, みる, おきる |
| 不規則 | Irregular | Exactly two verbs | する, くる |

### ます-Form: Four Variants (Ch 8)

| Variant | Formation (一段: drop る + ...) | Example (たべる) |
|---|---|---|
| Non-past affirmative | ます | たべます |
| Non-past negative | ません | たべません |
| Past affirmative | ました | たべました |
| Past negative | ませんでした | たべませんでした |

For 五段 verbs, shift the final kana to the い-row before adding ます (e.g., かく → かきます, のむ → のみます). Irregular: する → します, くる → きます.

### て-Form Formation (Ch 10)

| Verb type | Rule | Example |
|---|---|---|
| 一段 | Drop る, add て | たべる → たべて |
| 五段 ～く | く → いて | かく → かいて |
| 五段 ～ぐ | ぐ → いで | およぐ → およいで |
| 五段 ～す | す → して | はなす → はなして |
| 五段 ～つ/～る/～う | つ/る/う → って | まつ → まって |
| 五段 ～ぬ/～ぶ/～む | ぬ/ぶ/む → んで | のむ → のんで |
| 不規則 する | して | する → して |
| 不規則 くる | きて | くる → きて |
| Exception: いく | いって (not いいて) | いく → いって |

### ない-Form (Plain Negative) (Ch 12)

| Verb type | Rule | Example |
|---|---|---|
| 一段 | Drop る, add ない | たべる → たべない |
| 五段 | Shift final kana to あ-row, add ない | かく → かかない |
| 五段 ～う | う → わない | かう → かわない |
| 不規則 する | しない | する → しない |
| 不規則 くる | こない | くる → こない |
| Exception: ある | ない (not あらない) | ある → ない |

### た-Form (Plain Past) (Ch 10, 12)

The た-form follows the same sound-change rules as the て-form, replacing て with た and で with だ.

| Verb type | Rule | Example |
|---|---|---|
| 一段 | Drop る, add た | たべる → たべた |
| 五段 ～く | く → いた | かく → かいた |
| 五段 ～ぐ | ぐ → いだ | およぐ → およいだ |
| 五段 ～す | す → した | はなす → はなした |
| 五段 ～つ/～る/～う | つ/る/う → った | まつ → まった |
| 五段 ～ぬ/～ぶ/～む | ぬ/ぶ/む → んだ | のむ → のんだ |
| 不規則 する | した | する → した |
| 不規則 くる | きた | くる → きた |
| Exception: いく | いった | いく → いった |

### Dictionary Form Uses (Ch 8, 12)

The dictionary form (辞書形) is the plain non-past affirmative. It is the base form found in dictionaries and is required before the following grammar patterns:

| Context | Example |
|---|---|
| Before 前に ("before") | ねる 前に ほんを よみます。 |
| Before つもりです ("intend to") | にほんに いく つもりです。 |
| Before とおもいます ("I think") | あめが ふる とおもいます。 |
| Before でしょう ("probably") | あした あめが ふるでしょう。 |
| Before ことができる ("can") | にほんごを はなすことができます。 |
| In casual speech (sentence-final) | あした いく。 |

|---|---|---|---|

## Adjective Forms

### い-Adjectives (Ch 13)

All い-adjectives conjugate by replacing the final い with the appropriate ending. Stem = everything before the final い.

| Form | Formation | Example (たかい) |
|---|---|---|
| Non-past affirmative | stem + い (+ です) | たかいです |
| Non-past negative | stem + くない (+ です) | たかくないです |
| Past affirmative | stem + かった (+ です) | たかかったです |
| Past negative | stem + くなかった (+ です) | たかくなかったです |
| Prenominal (before noun) | い-adjective + noun | たかい やま |
| て-form (chaining) | stem + くて | たかくて おいしいです。 |

**いい/よい — Irregular:** いい conjugates from the stem よ: よくない, よかった, よくなかった, よくて. The dictionary form いい is the only form that does not use よ.

### な-Adjectives (Ch 14)

な-adjectives do not change form themselves. Conjugation uses the copula.

| Form | Formation | Example (しずか) |
|---|---|---|
| Non-past affirmative | stem + です | しずかです |
| Non-past negative | stem + じゃありません | しずかじゃありません |
| Past affirmative | stem + でした | しずかでした |
| Past negative | stem + じゃありませんでした | しずかじゃありませんでした |
| Prenominal (before noun) | stem + な + noun | しずかな まち |
| て-form (chaining) | stem + で | しずかで きれいです。 |

**Common traps:** きれい and ゆうめい end in い but are な-adjectives. They follow the な-adjective pattern (きれいじゃない, not *きれくない).

## Grammar Patterns

### Wants and Desires (Ch 19)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| たい | ます-stem + たい (conjugates as い-adj) | "Want to do" (speaker only) | にほんに いきたいです。 |
| たがる | ます-stem + たがる (conjugates as 五段 verb) | "Appears to want to do" (third person) | こどもが アイスクリームを たべたがっています。 |
| ほしい | [thing]が + ほしい (い-adjective) | "Want a thing" (speaker only) | あたらしい くるまが ほしいです。 |

### Intentions and Obligations (Ch 19)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| つもり | Plain non-past verb + つもりです | "Intend to" | らいねん、だいがくに はいる つもりです。 |
| なければならない | ない-form, drop い + ければならない | "Must do" | くすりを のまなければなりません。 |
| なくてはいけない | ない-form, drop い + くてはいけない | "Must do" (alternate) | しゅくだいを ださなくてはいけません。 |

Casual contractions: なければ → なきゃ, なくては → なくちゃ. Example: べんきょうしなきゃ。

### Advice and Conjecture (Ch 19)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| ほうがいい (affirmative) | た-form + ほうがいい(です) | "Had better do" | はやく ねたほうがいいですよ。 |
| ほうがいい (negative) | ない-form + ほうがいい(です) | "Had better not do" | あまり おさけを のまないほうがいいですよ。 |
| でしょう | Plain form + でしょう | "Probably" / conjecture | あした あめが ふるでしょう。 |
| とおもう | Plain form + とおもいます | "I think" | この ほんは おもしろい とおもいます。 |

Note: Before でしょう and とおもう, な-adjectives and nouns drop だ for でしょう (しずかでしょう) but keep だ for とおもう (しずかだ とおもいます).

### Invitations and Suggestions (Ch 20)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| ましょう | ます-stem + ましょう | "Let's" (proposal) | いっしょに ひるごはんを たべましょう。 |
| ましょうか | ます-stem + ましょうか | "Shall I?" (offer) | にもつを もちましょうか。 |
| ませんか | ます-stem + ませんか | "Won't you?" (polite invitation) | いっしょに コーヒーを のみませんか。 |

### Reasons and Contrast (Ch 20)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| から | Clause (plain or polite) + から | "Because" (direct, subjective) | あしたは しけんが ありますから、こんばん べんきょうします。 |
| ので | Plain form + ので (な-adj/noun: な + ので) | "Because" (softer, objective) | あめが ふっているので、でかけたくないです。 |
| けど / けれども | Clause + けど | "But" / "although" | あの レストランは おいしいですけど、ちょっと たかいです。 |
| Trailing けど | Clause + けど... (no conclusion) | Soft refusal / topic introduction | あしたは ちょっと いそがしいんですけど... |

### Explanatory んです (Ch 20)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| んです (statement) | Plain form + んです (な-adj/noun: な + んです) | "The thing is..." (explanation) | あたまが いたいんです。 |
| んですか (question) | Plain form + んですか | "Is it that...?" (seeking explanation) | どうしたんですか。 |

### Giving and Receiving (Ch 21)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| あげる | Giver は + Receiver に + Thing を + あげる | "Give" (away from speaker) | わたしは ともだちに プレゼントを あげました。 |
| もらう | Receiver は + Giver に/から + Thing を + もらう | "Receive" (toward receiver) | わたしは ともだちに ほんを もらいました。 |
| くれる | Giver は + (Speaker/in-group)に + Thing を + くれる | "Give" (toward speaker) | ともだちは わたしに おみやげを くれました。 |

### Time Clauses (Ch 25)

| Pattern | Formation | Meaning | Example |
|---|---|---|---|
| 前に | Dictionary form + 前に (noun: の前に) | "Before doing" | ねる 前に ほんを よみます。 |
| 後で | た-form + 後で (noun: の後で) | "After doing" | たべた 後で さんぽします。 |
| 時 (non-past) | Dictionary form + 時 | "When (about to do)" | 日本に いく 時 くうこうで おみやげを かいました。 |
| 時 (past) | た-form + 時 | "When (having done)" | うちに かえった 時 ねこが ねていました。 |
| 時 (い-adj) | い-adjective + 時 | "When [adjective]" | さむい 時 あたたかい ものを のみたいです。 |
| 時 (な-adj) | な-adjective + な + 時 | "When [adjective]" | ひまな 時 なにを しますか。 |
| 時 (noun) | Noun + の + 時 | "When [noun]" / "at the time of" | こどもの 時 やさいが きらいでした。 |
| ながら | ます-stem + ながら | "While doing" (same subject) | おんがくを ききながら べんきょうします。 |

Key rule for 前に and 後で: The verb before 前に is always dictionary form (the action has not yet happened). The verb before 後で is always た-form (the action is already complete). These rules hold regardless of the tense of the main clause.

### Numbers and Counters (Ch 7)

Numbers follow a transparent multiplication-and-addition system: 21 = にじゅういち (two-ten-one). Japanese counts objects using counters (助数詞) placed after the number.

| Counter | Used for | Example |
|---|---|---|
| ～つ | General objects (native Japanese, 1-9) | りんごを みっつ ください。 |
| ～人 (にん) | People (ひとり, ふたり, さんにん...) | がくせいが ごにん います。 |
| ～まい | Flat objects (paper, plates, shirts) | かみを にまい ください。 |
| ～ほん/ぽん/ぼん | Long, thin objects (pens, bottles, roads) | ペンを さんぼん かいました。 |
| ～さつ | Bound volumes (books, notebooks) | ほんを いっさつ よみました。 |
| ～だい | Machines, vehicles | くるまが にだい あります。 |
| ～ひき/ぴき/びき | Small animals | ねこが にひき います。 |

| Expression | Meaning | Example |
|---|---|---|
| ～じ | Hour (o'clock) | よじに おきます。(4:00) |
| ～ふん/ぷん | Minutes | じゅっぷんまえに きました。 |

### Sentence-Final Particles (Ch 23)

| Particle | Function | Example |
|---|---|---|
| よ | Informing (the listener does not know this) | おいしいですよ。 |
| ね | Confirming (the listener likely agrees) | いい てんきですね。 |
| よね | Informing + seeking agreement | あしたは にちようびですよね。 |

よ asserts new information to the listener. ね invites the listener to confirm shared knowledge. よね combines both: the speaker offers information while checking that the listener agrees.

### Question Words (Ch 5)

| Word | Meaning | Example |
|---|---|---|
| なに / なん | What | これは なんですか。 |
| だれ | Who | あの 人は だれですか。 |
| どこ | Where | 大学は どこですか。 |
| いつ | When | しけんは いつですか。 |
| どうして / なぜ | Why | どうして きませんでしたか。 |
| どう | How | にほんは どうですか。 |
| どの + noun | Which (of three+) | どの 大学が いいですか。 |
| どちら | Which (of two) / where (polite) | コーヒーと こうちゃと、どちらが すきですか。 |

Question words stay in the position where the answer would go. Japanese does not front question words.
