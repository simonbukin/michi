# Appendix D — Verb Conjugation Tables

Each verb is shown in all five forms covered: 辞書形 (dictionary form), ます-forms (affirmative, negative, past, past-negative), ない-form, た-form, and て-form. No new grammar is introduced here — this is purely for quick lookup.

### つくる (to make) — る-ending (五段)

| Form | Conjugation |
|------|------------|
| 辞書形 | つくる |
| ます | つくります |

| ませんでした | つくりませんでした |
| ない | つくらない |
| た | つくった |
| て | つくって |

This verb ends in る but is 五段, not 一段. The sound before る is in the う-row (くる), not in the い-row or え-row. This is exactly the kind of verb where class identification matters — conjugating it as 一段 would produce incorrect forms.

### いく (to go) — Exception

| Form | Conjugation |
|------|------------|
| 辞書形 | いく |
| ます | いきます |
| ません | いきません |
| ました | いきました |
| ませんでした | いきませんでした |
| ない | いかない |
| た | いった |
| て | いって |

いく is a く-ending 五段 verb and is regular in every form except た-form and て-form. A regular く-ending verb would give いいた and いいて; instead, いく uses いった and いって, following the つ/る/う pattern. This is the only く-ending verb with this irregularity. The ます-form and ない-form are entirely regular.

## D.3 不規則動詞 (Irregular Verbs)

Japanese has exactly two irregular verbs. Their forms must be memorized individually.

---

| Form | Conjugation |
|------|------------|

| ませんでした | きませんでした |

くる is the only verb in the language whose stem vowel changes across conjugations: く in the dictionary form, き in the ます-form and た/て-forms, こ in the ない-form. There is no pattern to extract — memorize the three shapes.

## D.5 音便 (Sound Change) Groupings

The た-form and て-form changes in 五段 verbs are driven by 音便 (おんびん) — euphonic sound changes. The nine endings fall into four groups:

| 音便 type | Endings | た-form pattern | て-form pattern |
|-----------|---------|----------------|----------------|
| イ音便 | く, ぐ | ～いた, ～いだ | ～いて, ～いで |
| 撥音便（はつおんびん） | ぬ, ぶ, む | ～んだ | ～んで |
| 促音便（そくおんびん） | つ, る, う | ～った | ～って |
| No change | す | ～した | ～して |

Note the voicing distinction within イ音便: く-ending verbs give いた/いて (unvoiced), while ぐ-ending verbs give いだ/いで (voiced). The voicing of the original ending carries through.

## D.6 Adjective Conjugation Summary

Adjectives are not verbs, but they conjugate in Japanese. Stage 1 covers the following forms for both types.

---

### い-adjective paradigm

Using たかい (expensive / tall) as the representative example:

| Form | Conjugation | Usage |
|------|------------|-------|
| 辞書形 (plain non-past) | たかい | Predicate or modifier, present/future |
| です form (polite non-past) | たかいです | Polite predicate, present/future |
| Past (plain) | たかかった | Plain past predicate |
| Past (polite) | たかかったです | Polite past predicate |
| Negative (plain) | たかくない | Plain negative predicate |
| Negative (polite) | たかくないです / たかくありません | Polite negative predicate |
| Past negative (plain) | たかくなかった | Plain past negative predicate |
| Past negative (polite) | たかくなかったです / たかくありませんでした | Polite past negative predicate |
| て-form | たかくて | Linking ("is expensive and...") |
| Adverb form | たかく | Modifying a verb |
| Modifier (before noun) | たかい + noun | たかいビル "a tall building" |

**Transformation rule:** remove い from the dictionary form to get the stem (たか～), then attach the appropriate ending.

| Target form | Ending attached to stem |
|-------------|----------------------|
| Non-past | ～い |
| Past | ～かった |
| Negative | ～くない |
| Past negative | ～くなかった |
| て-form | ～くて |
| Adverb | ～く |

**Exception:** いい (good) uses an irregular stem よ～ for all forms except the plain non-past:

| Form | Conjugation |
|------|------------|
| 辞書形 | いい |
| Past | よかった |
| Negative | よくない |
| Past negative | よくなかった |
| て-form | よくて |
| Adverb | よく |

---

### な-adjective paradigm

Using しずか (quiet) as the representative example:

| Form | Conjugation | Usage |
|------|------------|-------|
| 辞書形 (plain non-past) | しずかだ | Plain predicate, present/future |
| です form (polite non-past) | しずかです | Polite predicate, present/future |
| Past (plain) | しずかだった | Plain past predicate |
| Past (polite) | しずかでした | Polite past predicate |
| Negative (plain) | しずかじゃない / しずかではない | Plain negative predicate |
| Negative (polite) | しずかじゃないです / しずかじゃありません / しずかではありません | Polite negative predicate |
| Past negative (plain) | しずかじゃなかった / しずかではなかった | Plain past negative predicate |
| Past negative (polite) | しずかじゃなかったです / しずかじゃありませんでした / しずかではありませんでした | Polite past negative predicate |
| て-form | しずかで | Linking ("is quiet and...") |
| Modifier (before noun) | しずかな + noun | しずかなへや "a quiet room" |

**Transformation rule:** な-adjectives conjugate identically to nouns with the copula. The adjective stem itself never changes — only the copula (だ/です) conjugates.

| Target form | Pattern |
|-------------|---------|
| Non-past (plain) | ～だ |
| Non-past (polite) | ～です |
| Past (plain) | ～だった |
| Past (polite) | ～でした |
| Negative (plain) | ～じゃない |
| Negative (polite) | ～じゃありません |
| Past negative (plain) | ～じゃなかった |
| Past negative (polite) | ～じゃありませんでした |
| て-form | ～で |
| Noun modifier | ～な + noun |

## D.8 Quick-Reference: Identifying Verb Class

When encountering an unfamiliar verb in dictionary form, use this decision procedure:

1. **Is it する or くる?** → Irregular.
2. **Does it end in a noun + する?** (e.g., べんきょうする, さんぽする) → する-compound (irregular).
3. **Does it end in る with an い-row or え-row sound before る?** → Likely 一段. (But check for known 五段 exceptions like かえる "to return," はいる "to enter," はしる "to run," きる "to cut.")
4. **Everything else** → 五段.

This heuristic is reliable but not infallible. A small number of verbs ending in -いる or -える are 五段 rather than 一段. These must be learned individually. Dictionaries mark verb class; when in doubt, look it up.
