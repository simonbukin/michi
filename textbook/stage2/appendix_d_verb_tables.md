# Appendix D — Complete Verb Conjugation Tables

This appendix provides the full conjugation paradigm for all four verb classes, covering every form introduced in Stages 1 and 2. Model verbs: 書く (五段), 食べる (一段), する (irregular), 来る (irregular).

---

## How to Read These Tables

- **Dictionary form** = plain non-past affirmative (the "base" form found in dictionaries)
- All forms shown are in **plain style** unless marked (ます) for polite equivalents
- Negative forms end in ない (plain) or ません (polite)
- Past forms end in た/だ (plain) or ました (polite)
- Contracted/colloquial variants are noted where applicable

---

## Table 1: 五段動詞 — Model verb: 書く（かく）

| Form | Affirmative | Negative |
|------|------------|----------|
| **Dictionary (plain non-past)** | 書く | 書かない |
| **ます (polite non-past)** | 書きます | 書きません |
| **た (plain past)** | 書いた | 書かなかった |
| **ました (polite past)** | 書きました | 書きませんでした |
| **て-form** | 書いて | 書かなくて / 書かないで |
| **Potential** | 書ける | 書けない |
| **Volitional** | 書こう | — |
| **ましょう (polite volitional)** | 書きましょう | — |
| **Passive** | 書かれる | 書かれない |
| **Causative** | 書かせる | 書かせない |
| **Causative-passive (full)** | 書かせられる | 書かせられない |
| **Causative-passive (contracted)** | 書かされる | 書かされない |
| **Imperative** | 書け | — |
| **Prohibitive** | — | 書くな |
| **なさい (softened command)** | 書きなさい | — |
| **Conditional ば** | 書けば | 書かなければ |
| **Conditional たら** | 書いたら | 書かなかったら |

### 五段 て-form Sound Changes

The て-form and た-form of 五段 verbs undergo consonant changes depending on the final kana of the dictionary form:

| Dictionary ending | て-form | た-form | Example |
|------------------|---------|---------|---------|
| く | いて | いた | 書く → 書いて → 書いた |
| ぐ | いで | いだ | 泳ぐ → 泳いで → 泳いだ |
| す | して | した | 話す → 話して → 話した |
| つ | って | った | 待つ → 待って → 待った |
| う | って | った | 買う → 買って → 買った |
| る | って | った | 取る → 取って → 取った |
| ぬ | んで | んだ | 死ぬ → 死んで → 死んだ |
| ぶ | んで | んだ | 遊ぶ → 遊んで → 遊んだ |
| む | んで | んだ | 読む → 読んで → 読んだ |

**Exception:** 行く → 行って → 行った (not いいて)

## Table 2: 一段動詞 — Model verb: 食べる（たべる）

| Form | Affirmative | Negative |
|------|------------|----------|
| **Dictionary (plain non-past)** | 食べる | 食べない |
| **ます (polite non-past)** | 食べます | 食べません |
| **た (plain past)** | 食べた | 食べなかった |
| **ました (polite past)** | 食べました | 食べませんでした |
| **て-form** | 食べて | 食べなくて / 食べないで |
| **Potential** | 食べられる | 食べられない |
| **Potential (ら抜き)** | 食べれる | 食べれない |
| **Volitional** | 食べよう | — |
| **ましょう (polite volitional)** | 食べましょう | — |
| **Passive** | 食べられる | 食べられない |
| **Causative** | 食べさせる | 食べさせない |
| **Causative-passive** | 食べさせられる | 食べさせられない |
| **Imperative** | 食べろ | — |
| **Prohibitive** | — | 食べるな |
| **なさい (softened command)** | 食べなさい | — |
| **Conditional ば** | 食べれば | 食べなければ |
| **Conditional たら** | 食べたら | 食べなかったら |

**Note on ambiguity:** For 一段 verbs, the potential form (食べられる) and the passive form (食べられる) are identical. Context disambiguates. The colloquial ら抜き potential (食べれる) resolves this ambiguity, which is one reason for its widespread adoption in spoken Japanese.

## Table 3: する (irregular)

| Form | Affirmative | Negative |
|------|------------|----------|
| **Dictionary (plain non-past)** | する | しない |
| **ます (polite non-past)** | します | しません |
| **た (plain past)** | した | しなかった |
| **ました (polite past)** | しました | しませんでした |
| **て-form** | して | しなくて / しないで |
| **Potential** | できる | できない |
| **Volitional** | しよう | — |
| **ましょう (polite volitional)** | しましょう | — |
| **Passive** | される | されない |
| **Causative** | させる | させない |
| **Causative-passive** | させられる | させられない |
| **Imperative** | しろ | — |
| **Prohibitive** | — | するな |
| **なさい (softened command)** | しなさい | — |
| **Conditional ば** | すれば | しなければ |
| **Conditional たら** | したら | しなかったら |

**Note:** する compounds (勉強する, 運動する, etc.) follow the same pattern: 勉強する → 勉強できる → 勉強される → 勉強させる, etc. The noun portion remains unchanged; only する conjugates.

## Table 4: 来る（くる）(irregular)

| Form | Affirmative | Negative |
|------|------------|----------|
| **Dictionary (plain non-past)** | 来る（くる） | 来ない（こない） |
| **ます (polite non-past)** | 来ます（きます） | 来ません（きません） |
| **た (plain past)** | 来た（きた） | 来なかった（こなかった） |
| **ました (polite past)** | 来ました（きました） | 来ませんでした（きませんでした） |
| **て-form** | 来て（きて） | 来なくて（こなくて）/ 来ないで（こないで） |
| **Potential** | 来られる（こられる） | 来られない（こられない） |
| **Potential (ら抜き)** | 来れる（これる） | 来れない（これない） |
| **Volitional** | 来よう（こよう） | — |
| **ましょう (polite volitional)** | 来ましょう（きましょう） | — |
| **Passive** | 来られる（こられる） | 来られない（こられない） |
| **Causative** | 来させる（こさせる） | 来させない（こさせない） |
| **Causative-passive** | 来させられる（こさせられる） | 来させられない（こさせられない） |
| **Imperative** | 来い（こい） | — |
| **Prohibitive** | — | 来るな（くるな） |
| **なさい (softened command)** | 来なさい（きなさい） | — |
| **Conditional ば** | 来れば（くれば） | 来なければ（こなければ） |
| **Conditional たら** | 来たら（きたら） | 来なかったら（こなかったら） |

**Note on 来る readings:** The kanji 来 changes its reading depending on the form:
- く: dictionary form (来る), conditional ば (来れば), prohibitive (来るな)
- き: ます-stem forms (来ます, 来た, 来て, 来なさい)
- こ: ない-stem forms (来ない, 来よう, 来られる, 来させる, 来い)

## Notes on Contracted Forms

The following contractions are standard in spoken Japanese and widely used in informal writing:

| Full form | Contraction | Example |
|-----------|------------|---------|
| ている | てる | 食べている → 食べてる |
| ていた | てた | 食べていた → 食べてた |
| ておく | とく | 買っておく → 買っとく |
| ておいた | といた | 買っておいた → 買っといた |
| てしまう | ちゃう (after て) / じゃう (after で) | 食べてしまう → 食べちゃう / 飲んでしまう → 飲んじゃう |
| てしまった | ちゃった / じゃった | 忘れてしまった → 忘れちゃった |
| なければ | なきゃ | 行かなければ → 行かなきゃ |
| なくては | なくちゃ | 食べなくては → 食べなくちゃ |
| ては | ちゃ | 見てはいけない → 見ちゃいけない |
| では | じゃ | 遊んではいけない → 遊んじゃいけない |

These contractions apply uniformly across all verb classes.

## Potential Verbs Conjugate as 一段

Regardless of the original verb class, all potential forms conjugate as 一段 verbs:

| Original verb | Potential | Potential ます | Potential ない | Potential て |
|--------------|-----------|--------------|--------------|-------------|
| 書く (五段) | 書ける | 書けます | 書けない | 書けて |
| 読む (五段) | 読める | 読めます | 読めない | 読めて |
| 話す (五段) | 話せる | 話せます | 話せない | 話せて |

---

## Passive and Causative Verbs Also Conjugate as 一段

| Derived form | ます | ない | た | て |
|-------------|------|------|---|---|
| 書かれる (passive) | 書かれます | 書かれない | 書かれた | 書かれて |
| 書かせる (causative) | 書かせます | 書かせない | 書かせた | 書かせて |
| 書かせられる (caus.-pass.) | 書かせられます | 書かせられない | 書かせられた | 書かせられて |
