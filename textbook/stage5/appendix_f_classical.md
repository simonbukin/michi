# Appendix F — Classical Grammar Reference

This appendix is a comprehensive reference for the classical Japanese (文語) forms introduced in Stage 5 Chapters 10-11. Classical grammar is not merely historical — it permeates modern legal prose, formal writing, proverbs, literary allusion, and everyday set phrases. The goal here is recognition, not production: you should be able to identify these forms when they appear and understand their function in context.

For classical forms introduced in Stage 4 (～ざる, ～ず, ～べからず, ～ごとし, and the temporal patterns), see the Stage 4 Appendix F.

---

## Classical Verb Classes

Classical Japanese had more conjugation classes than modern Japanese. The following table maps each classical class to its modern descendant.

### 四段活用 → Modern 五段活用

The largest classical verb class. Four vowels (a, i, u, e) across conjugation forms; modern 五段 added the お-row for the volitional.

| 活用形 | 四段 (classical) | 五段 (modern) | Example: 書く |
|--------|-----------------|--------------|--------------|
| 未然形 | -a | -a | 書か |
| 連用形 | -i | -i | 書き |
| 終止形 | -u | -u | 書く |
| 連体形 | -u | -u | 書く |
| 已然形 | -e | (absorbed into 仮定形) | 書け |
| 仮定形 | — | -e | 書け |
| 命令形 | -e | -e | 書け |
| 未然形② | — | -o (volitional) | 書こ |

**Key difference:** Classical 已然形 + ば = "since/because" (established fact). Modern 仮定形 + ば = "if" (conditional). Same form, different meaning.

> 急がば回れ — Classical: 未然形 + ば = hypothetical "if one hurries"

### 上二段活用 / 下二段活用 → Modern 一段

Two classical classes that each used two vowels collapsed into single-vowel modern patterns.

| Type | Vowel alternation | Example | Modern form |
|------|-------------------|---------|-------------|
| 上二段 | i / u | 起く (おく) | 起きる (上一段) |
| 下二段 | e / u | 受く (うく) | 受ける (下一段) |

**What happened:** The 連体形 (-uru / -uru) replaced the 終止形 (-u) when the two forms merged in late medieval Japanese, then simplified to -iru / -eru.

| Classical 終止形 | Modern form | Meaning |
|-----------------|-------------|---------|
| 落つ (おつ) | 落ちる | to fall |
| 見ゆ (みゆ) | 見える | to be visible |
| 聞こゆ (きこゆ) | 聞こえる | to be audible |

### カ行変格活用 (カ変) — 来

Irregular in both classical and modern Japanese.

| 活用形 | Classical | Modern |
|--------|-----------|--------|
| 未然形 | こ | こ |
| 連用形 | き | き |
| 終止形 | く | くる |
| 連体形 | くる | くる |
| 已然形 | くれ | くれ |
| 命令形 | こ / こよ | こい |

### サ行変格活用 (サ変) — す (modern する)

| 活用形 | Classical | Modern |
|--------|-----------|--------|
| 未然形 | せ | し / せ / さ |
| 連用形 | し | し |
| 終止形 | す | する |
| 連体形 | する | する |
| 已然形 | すれ | すれ |
| 命令形 | せよ | しろ / せよ |

The classical 未然形 せ produces the irregular forms せず, せざる, せざるを得ない (not しず, しざる).

### ナ行変格活用 (ナ変) — 死ぬ, 往ぬ (去ぬ)

Two verbs only. 死ぬ was absorbed into modern 五段 conjugation. 往ぬ/去ぬ (いぬ, "to go away") is archaic.

### ラ行変格活用 (ラ変) — あり, をり, はべり, いまそかり

Four verbs whose defining peculiarity was a 終止形 ending in -り instead of the expected -る.

| Verb | Meaning | Register |
|------|---------|----------|
| あり | to be, to exist | Neutral |
| をり | to be, to sit | Neutral (modern おる) |
| はべり | to serve, to be (humble) | Humble/polite |
| いまそかり | to be (honorific) | Honorific |

> あり → 終止形: あり (not ある)

This explains why the classical copula なり ends in -り: it derives from に + あり. Modern ある regularized the 終止形 to match the 連体形.

## Classical Auxiliary Verbs in Modern Set Phrases

### ず — Negative

The classical negative auxiliary. Attached to the 未然形 (irrealis stem) of verbs.

#### Conjugation of ず

| 活用形 | Form |
|--------|------|
| 未然形 | ず |
| 連用形 | ず |
| 終止形 | ず |
| 連体形 | ぬ / ざる |
| 已然形 | ね / ざれ |

#### Modern set phrases preserving ず

| Phrase | Reading | Meaning | Analysis |
|--------|---------|---------|----------|
| やむを得ず | やむをえず | unavoidably | やむ + を + 得 + ず ("cannot obtain cessation") |
| 知らず知らず | しらずしらず | unknowingly | Doubled ず form: "not knowing, not knowing" |
| 絶えず | たえず | constantly, incessantly | 絶え + ず ("without ceasing") |
| 少なからず | すくなからず | not a little, considerably | 少なから + ず (classical negative of くあり) |
| 限らず | かぎらず | not limited to | 限ら + ず |
| 問わず | とわず | regardless of | 問わ + ず ("without asking") |
| 尽きず | つきず | inexhaustibly | 尽き + ず ("without running out") |
| せず | せず | without doing | せ (未然形 of す) + ず |
| 予期せぬ | よきせぬ | unexpected | せ + ぬ (連体形 of ず) |

### たり — Perfective

Indicated a completed action or resulting state. The 連体形 たる survives in modern Japanese to create dignified adjective-like expressions from 漢語.

#### Surviving forms

| Form | Function | Example |
|------|----------|---------|
| たる (連体形) | Modifies nouns | 堂々たる態度 |
| たり (終止形) | Ends sentences (rare, literary) | 堂々たり |

#### Modern set phrases preserving たる

| Phrase | Reading | Meaning |
|--------|---------|---------|
| 堂々たる | どうどうたる | dignified, imposing |
| 泰然たる | たいぜんたる | composed, calm |
| 断固たる | だんこたる | resolute, firm |
| 悠々たる | ゆうゆうたる | leisurely, unhurried |
| 厳然たる | げんぜんたる | stern, solemn |
| 整然たる | せいぜんたる | orderly, systematic |

The pattern is: 漢語 + たる + noun. This produces a formal, elevated tone that 漢語 + な cannot achieve.

### べし — Obligation / Conjecture

Expressed obligation ("should"), strong conjecture ("must be"), or fitness ("it is fitting to"). Attached to the 終止形 of verbs.

#### Surviving forms

| Form | Function | Example |
|------|----------|---------|
| べき (連体形) | Modifies nouns: "should" | すべきこと |
| べく (連用形) | Adverbial: "in order to" | 知るべく |
| べし (終止形) | Sentence-final (literary) | 知るべし |
| べからず (べし + ず) | Prohibition: "must not" | 入るべからず |
| べからざる (べからず attributive) | "Must not" before nouns | 許すべからざる行為 |

#### Modern set phrases preserving べし / べき

| Phrase | Reading | Meaning |
|--------|---------|---------|
| すべき | すべき | should do |
| あるべき姿 | あるべきすがた | the form something should take |
| 知るべし | しるべし | one should know (literary) |
| なすべきこと | なすべきこと | things that ought to be done |
| 入るべからず | はいるべからず | must not enter |

---

### まじ — Negative of べし

The negative counterpart of べし: "should not," "must not," "probably not." The 連体形 まじき survives in modern set phrases.

#### Modern set phrases preserving まじき

| Phrase | Meaning | Example |
|--------|---------|---------|
| あるまじき | that should not exist; unbecoming | あるまじき行為 ("behavior that should not exist") |
| 教師にあるまじき | unbecoming of a teacher | 教師にあるまじき発言だ。 |
| 許すまじき | unforgivable | 許すまじき犯罪 ("an unforgivable crime") |

---

### なり — Assertion / Hearsay

Classical Japanese had two homophonous なり with different functions:

| なり | Function | Attached to | Example |
|------|----------|------------|---------|
| 断定のなり | Copula: "is" | Nouns, 連体形 | 春なり。("It is spring.") |
| 伝聞推定のなり | Hearsay: "I hear that..." | 終止形 | 男来るなり。("I hear a man is coming.") |

#### Modern traces

| Usage | Example | Register |
|-------|---------|----------|
| Formal assertion | 事実なり。("It is a fact.") | Legal, literary |
| Attributive なる | 多大なる貢献 ("great contribution") | Formal set phrase |
| Archaic declaration | これぞ武士の道なり。 | Literary quotation |

---

### けり — Exclamatory Past

Had two functions: narrating past events and expressing emotional exclamation (sudden realization or deeply felt awareness).

#### Examples

| Usage | Example | Function |
|-------|---------|----------|
| Emotional exclamation | 春が来にけり。("Ah, spring has come.") | Poetic realization |
| Narrative past | 昔々...住んでおりけり。("Long ago...there lived...") | Fairy-tale convention |

～けり as a literary ending is a marker of the classical storytelling register. It is not interchangeable with modern past tense た.

## Classical Particles — 係り結び

係り結び (かかりむすび) was a system where emphatic or interrogative particles forced the sentence-final verb into a specific conjugation form. This system collapsed in the medieval period, but its traces survive in modern Japanese.

### The particles and their required verb forms

| 係助詞 | Meaning | Final verb form |
|--------|---------|----------------|
| ぞ | Emphasis | 連体形 |
| なむ | Emphasis | 連体形 |
| や | Question | 連体形 |
| か | Question | 連体形 |
| こそ | Strong emphasis | 已然形 |

### Examples

> 花ぞ咲ける。
> "The flowers ARE blooming." (ぞ forces 連体形)

> 我こそ知れ。
> "It is I who knows." (こそ forces 已然形)

> いづれの御時にか...候ひ給ひける。(源氏物語)
> "In which reign was it..." (か forces 連体形)

### Modern survivors

| Expression | Classical origin | Modern meaning |
|------------|-----------------|----------------|
| だからこそ | こそ emphasis | "Precisely because..." |
| ～てこそ | こそ emphasis | "Only by doing ~" |
| まさか | まさ + か (interrogative) | "No way, surely not" |
| いかが | Interrogative | "How about...?" |
| なぜ | 何ぞ + え → なぜ | "Why?" |

## Classical Copula なり and Modern Descendants

### Conjugation of classical なり

| 活用形 | Form |
|--------|------|
| 未然形 | なら |
| 連用形 | なり / に |
| 終止形 | なり |
| 連体形 | なる |
| 已然形 | なれ |
| 命令形 | なれ |

### Evolution into modern forms

| Modern form | Origin | Register |
|-------------|--------|----------|
| だ | Contraction of である (で + ある) | Colloquial |
| です | Politeness layer added to だ | Polite |
| である | で (連用形 of にてあり) + ある | Written, formal |
| なり | Direct survival | Very formal, literary |
| なる | Attributive form survival | Set phrases: 多大なる, 偉大なる |
