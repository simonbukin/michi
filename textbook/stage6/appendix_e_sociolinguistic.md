# Appendix E — Sociolinguistic Variation Reference

> Quick-reference for gendered markers, generational features, and dialect features (Chapters 6-8).

## 1. Gendered Speech Markers

### Sentence-Final Particles

| Particle | Traditional Association | Modern Usage |
|---|---|---|
| わ (rising) | Feminine | Declining; some male speakers in Kansai |
| わよ | Feminine | Mostly older female speakers |
| わね | Feminine | Mostly older female speakers |
| の (falling, declarative) | Feminine | Women in casual speech; men avoid or add だ |
| のよ | Feminine | Assertive feminine; declining among youth |
| かしら | Feminine | "I wonder"; men use かな instead |
| ぜ | Masculine | Assertive; casual male speech |
| ぞ | Masculine | Strong assertion/warning; rough male speech |
| な (self-directed) | Masculine-leaning | Increasingly gender-neutral among youth |
| よね | Neutral | Confirmation-seeking; all genders |
| じゃん | Neutral | Originally Kanto; now gender-neutral nationwide |

### Personal Pronouns

| Pronoun | Register | Gender Association | Notes |
|---|---|---|---|
| 私 (わたし) | Polite | Neutral | Standard polite for all genders |
| 私 (わたくし) | Very formal | Neutral | Business, ceremonies |
| 僕 (ぼく) | Casual | Male-leaning | Some women use it; common in songs |
| 俺 (おれ) | Casual | Strongly male | Rough/intimate; never in formal settings |
| あたし | Casual | Feminine | Softer variant of わたし |
| あたくし | Formal | Feminine | Refined; rare in modern speech |
| うち | Casual | Feminine / Kansai neutral | Widespread in Kansai for all genders |
| 自分 (じぶん) | Varies | Neutral | Reflexive or 2nd person in Kansai/military |
| わし | Casual | Male (older) | Elderly men; also Hiroshima/Kansai dialect |

### Vocabulary Choices

| Meaning | Feminine-leaning | Masculine-leaning | Neutral |
|---|---|---|---|
| Delicious | おいしい | うまい | おいしい |
| Very | とても、すごく | めちゃ、超 | すごく |
| Stomach | おなか | はら (腹) | おなか |
| To eat (casual) | 食べる | 食う (くう) | 食べる |
| This (thing) | これ | こいつ | これ |
| Honorific prefix | お〜 (frequent) | お〜 (less frequent) | Context-dependent |

### Conjugation Patterns

| Pattern | Feminine Tendency | Masculine Tendency |
|---|---|---|
| Imperative form | Avoided; uses ~て / ~てください | Direct imperative (行け、しろ) |
| Negative imperative | ~ないで | ~な (食うな) |
| Volitional | ~ましょう / ~ない？ | ~しよう / ~しようぜ |
| Copula (casual) | なの、~よ | だ、だぜ、だぞ |
| Tag questions | ~でしょう？ / ~わよね？ | ~だろう？ / ~じゃないか？ |

## 2. Generational Features

### Youth Slang (若者言葉)

| Expression | Meaning | Usage Notes |
|---|---|---|
| マジ | Seriously, really | Adverb or adjective; replaces 本当に |
| ヤバい | Amazing / terrible / extreme | Positive or negative depending on context |
| エモい | Emotional, moving, nostalgic | From English "emotional"; aesthetic appreciation |
| 推し (おし) | Favorite person/idol to support | From 推す; fandom culture |
| 沼 (ぬま) | Deep obsession, "swamp" | 沼にハマる = to fall deep into a hobby/fandom |
| ガチ | For real, serious, legit | ガチで = seriously; ガチ勢 = hardcore fans |
| それな | "That's so true" | Agreement marker; conversation filler |
| わかりみ | "I feel that" / relatability | み-nominalization trend (つらみ、うれしみ) |
| 草 (くさ) | LOL, funny | From wwww resembling grass; 草生える |
| ～み | Nominalizer for feelings | わかりみ、つらみ、やばみ; playful/informal |
| りょ / り | Roger, understood | Shortened from 了解 (りょうかい) |
| ワンチャン | Slight chance, maybe | From "one chance"; ワンチャンある |

### Older Speech Patterns (年配の方の言葉遣い)

| Feature | Example | Modern Equivalent |
|---|---|---|
| ござる・ございます (frequent) | さようでございますか | そうですか |
| ~でおじゃる | (archaic, now humorous) | ~です |
| ~たまえ (command) | 読みたまえ | 読んでくれ |
| おる (humble/neutral) | ここにおります | ここにいます |
| ~なさる (honorific) | どうなさいますか | どうしますか |
| じゃ (copula) | それじゃのう | それじゃね |
| わし (1st person) | わしが若い頃は | 俺が若い頃は |
| 拙者 (せっしゃ) | (samurai speech, now humorous) | 私 |

## 3. 関西弁 (Kansai Dialect)

### Verb Forms

| Standard | 関西弁 | Example (Standard → Kansai) |
|---|---|---|
| ~ない (negative) | ~へん / ~ん | 食べない → 食べへん |
| ~ている | ~てる / ~とる | 食べている → 食べとる |
| ~てしまう | ~てまう / ~てもうた (past) | 食べてしまった → 食べてもうた |
| ~ては (conditional) | ~たら / ~てもうたら | — |
| ~なければ | ~なあかん / ~んと | 行かなければ → 行かなあかん |
| いる | おる | ここにいる → ここにおる |
| ダメ | あかん | ダメだ → あかんで |

### Adjective Forms

| Standard | 関西弁 | Example |
|---|---|---|
| ~くない | ~ない | おもしろくない → おもろない |
| おもしろい | おもろい | — |
| すごい | えらい / めっちゃ | めっちゃおもろい |
| つまらない | おもんない | — |
| こわい | こわい / おっかない | — |

### Copula and Particles

| Standard | 関西弁 | Notes |
|---|---|---|
| だ / です | や / やで | 学生だ → 学生や |
| でしょう | やろう | そうでしょう → そうやろう |
| ではない | ちゃう / やない | 違う → ちゃう |
| ~のだ | ~ねん / ~のや | 行くんだ → 行くねん |
| ~から (reason) | ~さかい / ~から | さかい is traditional; declining |
| ~けれど | ~けど / ~やけど | — |
| とても | めっちゃ / ごっつ | ごっつ is stronger/older |
| 本当に | ほんまに | ほんまやで |

### Key Vocabulary

| Standard | 関西弁 | Meaning |
|---|---|---|
| ありがとう | おおきに | Thank you |
| 捨てる | ほかす | To throw away |
| しんどい | しんどい (origin) | Tiring, tough |
| 片付ける | なおす | To put away, tidy up |
| 大丈夫 | かまへん | It's fine, no problem |

## 4. 東北弁 (Tohoku Dialect) — Recognition Features

| Feature | Example | Standard Equivalent |
|---|---|---|
| Vowel merging: い/え collapse | えぐ (いく) | 行く |
| Voicing of か行 → が行 | かぎ → がぎ | 柿 (かき) |
| ず/じ merge (ズーズー弁) | すず → すじ (blurred) | — |
| ~だべ / ~べ (conjecture) | 行くべ | 行くだろう |
| ~っけ (confirmation) | そうだっけ？ | そうだっけ？ (adopted into standard) |
| け (question particle) | 行くけ？ | 行くの？ |
| ~さ (direction particle) | 学校さ行く | 学校に行く |
| んだ (affirmation) | んだ、んだ | そうだ、そうだ |
| Contracted negatives | わがんね | わからない |
